## 📋 구성 요약

### 1. 메인 화면 (Desktop · 1300px 이상)
에디터 최상단 고정 툴바. 높이 50px 고정.
- **좌측 A**: 문서 정보 (뒤로가기, 문서번호, 점수 상태)
- **우측 B**: 편집 도구 (되돌리기/다시하기, 캔버스 폭, 줌)
- **우측 C**: 주요 액션 (미리보기, 다운로드)

### 2. 반응형 UI 변화
- **L (< 1300)** / **M (900–1299)** / **S (768–899)** / **XS (≤ 767)**
- 중요 우선순위: 문제 정보/편집은 유지 → 점수 상태 → 공간 부족 시 드롭다운
- XS 전환: 페이지 구조 표시 (Todo)

### 3. 연관 팝업
| 팝업명 | 트리거 | 용도 |
| :--- | :--- | :--- |
| `title_edit_popup` | 관리명/이동명 영역 클릭 | 문서명 · 적용명 편집 |
| `state_view` | 점수 상태 클릭 | 점수 전 / 점수 중 / 점수 완료 선택 |
| `percent_view` | 줌 100% 클릭 | 20–100% 미리 설정된 배율 선택 |
