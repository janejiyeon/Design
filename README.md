## ✏️ 편집 가이드

GitHub 웹에서 바로 수정 가능합니다:
1. MD 파일 열기 → 우측 상단 연필 아이콘(✎) 클릭
2. 표 · 리스트 · 텍스트 편집 후 `Commit changes` 클릭
3. 목업 이미지는 `screenshots/` 폴더의 파일을 교체

> 💡 CSS 토큰, 반응형 테이블, 정책 목록 등 모든 텍스트 콘텐츠가 그대로 마크다운 표/리스트로 변환되어 있어 기획자/개발자가 각자 편집 가능합니다.
