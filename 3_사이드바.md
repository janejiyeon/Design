## 📋 구성 요약

### 1. 사이드바 탭 레일 (Desktop · 1300px 이상)
에디터 좌측 고정 탭 레일 (최소 64×974).
- 캔버스 스크롤 · 사이드 패널 개폐와 무관하게 항상 고정
- 진입 시 아무 탭도 선택되지 않은 상태가 디폴트
- 탭 구성: 총 10개 · 2개 그룹 (상단 8개 + 하단 2개)

### 탭 그룹
| 그룹 | 탭 목록 |
| :--- | :--- |
| **상단 (8개)** | 템플릿 · 임포트 · 믹스트 · 페이지 · 요소 · 북마크 · (구분선) · AI이미지 · AI기능 |
| **하단 (2개)** | 레이어 · 가이드 |

### 세부 정책

| # | 카테고리 | 주요 정책 |
| :--- | :--- | :--- |
| ① | 탭 레일 · 탭 1개 단위 | 64×64, 아이콘 18×18 + 캡션 11px Medium |
| ② | 탭 선택 · Hover 상태 | hover 효과 / selected 시 패널 확장 / 한 번에 1개 탭만 선택 가능 |
| ③ | 업로드 진행 배지 | 이미지 업로드 시작 시 우상단 Lottie 애니메이션 표시 |
| ④ | 스크롤 동작 | 탭 레일 높이가 뷰포트보다 클 경우 내부 수직 스크롤 |
| ⑤ | 다크모드 | 배경 `#181b1e` / 탭 아이콘 · 텍스트 토큰 정의 |

> 💡 **Tip**: 더블클릭으로 믹스트 편집 · 드래그로 번호/영역 이동 · ⌘S로 저장
